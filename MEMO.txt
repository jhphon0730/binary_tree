# - 두 커플의 공유 프로필 사진 수정하는 기능 추가 [ 백엔드 및 프론트 및 DB ]

3. 다이어리 기능 구현:
1. 일기 작성, 조회, 수정, 삭제 (CRUD) 기능
3. 공유 일기 작성 기능 (두 사람이 함께 작성이 가능함(A가작성하면A도수정,B도수정 가능))

4. 캘린더 기능:
1. 월별, 주별 캘린더 뷰
2. 일정 추가, 수정, 삭제 기능
3. 기념일 표시 기능

5. 사진 갤러리:
1. 사진 업로드 및 관리 기능
2. 앨범 생성 및 정리 기능

6. 메시지 또는 채팅 기능:
1. 실시간 또는 비실시간 메시지 교환 기능

7. 알림 시스템:
1. 중요 일정, 기념일, 새 메시지 등에 대한 알림 기능

8. 통계 및 분석:
1. 관계 기간, 만난 횟수, 특별한 날 등의 통계 제공

9. 설정 페이지:
1. 앱 테마 변경, 알림 설정, 개인정보 설정 등

10. 검색 기능:
1. 일기, 일정, 사진 등을 검색할 수 있는 기능


1. 사용자 프로필 페이지
2. 커플 연결 기능
3. 다이어리 CRUD 기능
4. 간단한 캘린더 기능

---------------------------

앱의 기능과 사용자 경험을 고려하여 페이지를 다음과 같이 구조화

#. 대시보드 (/dashboard)
0. 공유 프로필 사진 수정 기능 추가 [ 백엔드 및 프론트 및 DB ]
[OK]. 커플 공유 정보 표시 및 수정 (연애 시작일)
[OK]. 공유 메시지 표시 및 수정
3. 간단한 캘린더 위젯 (이번 달 주요 일정)
4. 최근 일기 미리보기
5. 최근 업로드된 사진 미리보기

#. 프로필 페이지 (/profile)
1. 사용자 정보 표시 및 수정 [ OK ]
2. 커플 사용자 정보 표시 [ OK ]
3. 공유 프로필 사진 수정

#. 다이어리 페이지 (/diary)
1. 일기 목록 표시
2. 일기 작성 버튼
3. 날짜별 일기 조회 기능
4. 서브 페이지:
	#. 일기 상세 보기 (/diary/[id])
	#. 일기 작성/수정 (/diary/edit/[id])

#. 캘린더 페이지 (/calendar)
1. 월별, 주별 캘린더 뷰
2. 일정 추가, 수정, 삭제 기능
3. 기념일 표시
4. 서브 페이지:
	#. 일정 상세 보기 및 수정 (/calendar/event/[id])

#. 갤러리 페이지 (/gallery)
1. 사진 그리드 뷰
2. 앨범 목록
3. 서브 페이지:
	#. 앨범 상세 보기 (/gallery/album/[id])
	#. 사진 상세 보기 및 수정 (/gallery/photo/[id])

#. 메시지 페이지 (/messages)
1. 메시지 목록
2. 메시지 작성 인터페이스

#. 설정 페이지 (/settings)
1. 앱 테마 변경
2. 알림 설정
3. 개인정보 설정
4. 통계 및 분석 보기

#. 검색 페이지 (/search)
1. 통합 검색 인터페이스 (일기, 일정, 사진 등)
